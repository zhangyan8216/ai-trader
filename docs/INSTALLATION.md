# Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/zhangyan8216/ai-trader.git
   cd ai-trader
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python app.py
   ```

# Troubleshooting Guide

- **Problem:** Application not starting.
  - **Solution:** Ensure all dependencies are installed and you're using the correct Python version.

- **Problem:** Module not found error.
  - **Solution:** Check if the module is listed in `requirements.txt` and installed properly.

- **Problem:** Unexpected behavior.
  - **Solution:** Check the logs for error messages and refer to the documentation for configuration issues.
